Business Logic & Progress Calculations

Achievement Progression Algorithm

function getAchievementLevel(totalActivities) {
  if (totalActivities >= 100) return 'Master';
  if (totalActivities >= 50) return 'Expert';
  if (totalActivities >= 25) return 'Advanced';
  if (totalActivities >= 10) return 'Intermediate';
  if (totalActivities >= 1) return 'Starter';
  return 'Beginner';
}

Subscription Status Logic

function getSubscriptionStatus(streak, total) {
  if (streak >= 7 || total >= 10)
    return 'premium_earned';
  if (streak >= 3 || total >= 7)
    return 'almost_premium';
  return 'free';
}

Daily Completion Rate Formula

Automatically calculated and stored in daily_trackers table:

completion_rate = (completed_entries / total_planned_entries) × 100

Updates in real-time as users mark activities as completed or skipped.

Streak Calculation Process

Counts consecutive days working backwards from today:

WITH consecutive_days AS (
  SELECT date, completion_rate
  FROM daily_trackers
  WHERE user_id = ? AND completion_rate > 0
  ORDER BY date DESC
)
SELECT COUNT(*) as current_streak
