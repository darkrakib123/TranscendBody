🔗 Table Relationships

  • users → daily_trackers (One-to-Many): A user can have multiple daily tracking sessions
  • daily_trackers → tracker_entries (One-to-Many): Each daily tracker contains multiple activity entries
  • activities → tracker_entries (One-to-Many): Each activity can be tracked multiple times across different sessions
  • sessions → Independent: Manages user authentication sessions

🏆 Achievement Levels
  • 🥉 Beginner (0-24 activities) - Bronze Theme
  • 🥈 Intermediate (25-49 activities) - Silver Theme
  • 🥇 Master (50+ activities) - Gold Theme
💎 Subscription Tiers
  • 🆓 Free Plan - Getting started (Bronze)
  • ⭐ Almost Premium - 7+ streak, 20+ activities (Silver)
  • 👑 Premium Earned - 14+ streak, 40+ activities (Gold)
📊 Key Metrics
  • 🔥 Daily Streak - Consecutive active days

  • 🎯 Total Activities - Lifetime completions

🔐 User Management
  • 👨‍💼 Admin - Full system access and user management

  • 🔑 Authentication - Secure login with sessions
  • 📊 Role-based features and permissions

🚀 Technical Implementation

  • Backend: Express.js with TypeScript for type safety
  • Database: PostgreSQL with Drizzle ORM for type-safe queries
  • Authentication: Session-based with secure cookie management
  • Frontend: Dynamic JavaScript with Bootstrap styling
  • Theming: CSS custom properties for bronze/silver/gold progression
  • Real-time: Dynamic subscription status and achievement calculation
